# Prometheus-Reverse
Welcome to the Prometheus-Reverse Project!
This project is a tool for reversing:
* The Vmify Preset (Work in Progress)
* The EncryptStrings Preset (Use DecryptStrings, fully implemented!)
* The ConstantArray Preset (Use ReverseConstantArray, fully implemented!)

## NOTE
The steps only run on LuaJIT. This is because of Luraph. Sorry!
